# Stock Price Prediction Using ARIMA

A time series forecasting project that predicts stock prices using the ARIMA (AutoRegressive Integrated Moving Average) model. This project analyzes historical stock market data, performs preprocessing and stationarity checks, and generates future stock price forecasts with visualization.

## Features

- Historical stock price analysis
- Data preprocessing and cleaning
- Stationarity testing using ADF test
- ACF and PACF analysis
- ARIMA model training and tuning
- Future stock price forecasting
- Visualization of actual vs predicted prices
- Performance evaluation using RMSE and MAPE

## Technologies Used

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Stock-Price-Prediction-Using-ARIMA.git
```

Move to the project directory:

```bash
cd Stock-Price-Prediction-Using-ARIMA
```

Install required dependencies:

```bash
pip install -r requirements.txt
```

## Usage

Run the project:

```bash
python app.py
```

or

```bash
python main.py
```
